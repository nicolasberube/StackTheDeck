conda activate StackTheDeck
pip install pygame

pip install pyinstaller
pyinstaller StackTheDeck.spec

pip install cx_freeze
python cx_setup.py bdist_msi

compile the C in .dll with visual studio





compute_preflop.py : computes preflop.pkl
cx_setup.py: cx_freeze setup file installer for Windows. Use with
	python cx_setup.py bdist_msi
freesansbold.ttf: Default font for pygame. Optional file.
holes_operations_NT.c: external C code for Windows. Must be compiled in a .dll with Visual Studio
holes_operations.c: external C code for Mac. Must be compiled in a .so with python so_setup.py
holes_operations.dll: C library for Windows
holes_operations.so: C library for Mac
preflop.pkl: pre flop probabilities data
StackTheDeck.py: Main game code
StackTheDeck-0.1-amd64.msi: cx_freeze game installer for Windows
StackTheDeck.exe: PyInstaller standalone .exe for Windows
StackTheDeck.spec: PyInstaller setup file for Windows. Needs setuptools<45 to work properly. Use with
	pyinstaller StackTheDeck.spec
